Как работать с файлами
1. В Qt Creator нужно установить режим сборки Release (по умолчанию - Debug);
2. Откройте проводник в месте, где лежит папка вашего проекта. Рядом с ней будет папка XXX-build-desktop-Qt... Откройте её.
3. В каталоге release будет финальная версия вашей программы, которую можно переносить на другие устройства. Сюда нужно добавить .dll файлы из репозитория с соответствующей версией Qt.
4. Готовый каталог release можно передавать как архив. В нём содержатся все зависимости, на которые ссылается ваша программа (если в ней нет файлов ресурсов).
